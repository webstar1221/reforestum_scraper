# Python Scraper For Reforestum
***

Collect the project list and details from react-based SPA website

Key Technologies:

- Python 3.10

- BeautifulSoup

- Selenium

# How to run

1. Create a virtual environment and install the dependencies:

    ```sh
    $ pipenv shell
    $ pipenv install
    ```

1. Run the scraper:

    ```sh
    $ python scraper.py
    ```
